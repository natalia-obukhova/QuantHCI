Basic keyboard shortcuts

To get <- : Alt+++-

Create a new script: Cmd/Ctrl + Shift + N

Execute the current expression: Cmd/Ctrl + Enter

Execute/rerun the complete script: Cmd/Ctrl + Shift + S

Restart R Studio: Cmd/Ctrl + Shift + F10

Press F to pay respect
